# Boston Experience 2026 — SERAC

Interactive digital travel brochure for the SERAC immersion program in Cambridge & Boston, April 15–17, 2026.

## Live Site

**[emilezounon.com/SERAC2026](https://emilezounon.com/SERAC2026)**

## About

A premium single-page travel pamphlet covering a 3-day program across Google, MIT, Harvard, the Broad Institute, and Boston's innovation ecosystem. Includes lectures, workshops, campus tours, networking events, and dining recommendations.

| Day | Date | Theme |
|-----|------|-------|
| 1 | April 15 | Google Cambridge & MIT Media Lab |
| 2 | April 16 | Museum of Science, Harvard & Venture Cafe |
| 3 | April 17 | Broad Institute, MIT Sloan & Sailing |

## Tech

- Single HTML file — no build tools, no frameworks
- Google Fonts (Playfair Display + Inter) — only external dependency
- CSS scroll animations via Intersection Observer
- Responsive design (mobile + desktop)
- All content in Portuguese (as originally written)
